================================================================================
shatter
================================================================================

.. only:: html

    Shatter inserts point cloud data into a SilviMetric database

.. Index:: shatter

Synopsis
--------

.. code-block::

    Usage: silvimetric DATABASE shatter [OPTIONS] POINTCLOUD

    Insert data provided by POINTCLOUD into the silvimetric DATABASE

    Options:
    --workers INTEGER
    --bounds BOUNDS
    --tilesize INTEGER
    --threads INTEGER
    --watch
    --dasktype [cluster|threads|processes|single-threaded]
    --help                          Show this message and exit.
